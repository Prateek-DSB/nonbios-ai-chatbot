## Quick Start

1. Clone:
```bash
git clone https://github.com/Prateek-DSB/nonbios-ai-chatbot.git
cd nonbios-ai-chatbot
```

2. Install:
```bash
npm install
```

3. Add `.env.local`:
```env
OPENAI_API_KEY=your_key_here
```

4. Run:
```bash
npm run dev
```
